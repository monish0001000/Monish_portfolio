# Agent Instructions

## Image Protection Rule
- **CRITICAL**: Do NOT modify, replace, or overwrite any files in the `/images` directory unless explicitly requested by the user.
- Any "beautification" or "refresh" of visual assets should be restricted to CSS/HTML changes, not direct binary file modifications in `/images`.
- If you feel an image needs updating, ASK the user first instead of using image generation tools to overwrite existing assets.
- This rule is set to prevent unwanted automatic modifications observed in previous turns where AI agents "refreshed" existing image assets without explicit instruction.
